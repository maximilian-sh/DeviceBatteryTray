                    DeviceBatteryTray Installation Guide
═══════════════════════════════════════════════════════════════════════════════

INSTALLATION
───────────────────────────────────────────────────────────────────────────────

Option 1: Automated Installer (Recommended)
───────────────────────────────────────────────────────────────────────────────

1. Extract this ZIP file to any temporary location (e.g., your Downloads folder)

2. Double-click "install.bat" to run the installer:
   - **For current user install:** Just double-click `install.bat` (no admin needed)
     - Installs to: %LOCALAPPDATA%\DeviceBatteryTray
   - **For system-wide install:** Right-click `install.bat` → Run as Administrator
     - Installs to: C:\Program Files\DeviceBatteryTray

4. The installer will:
   ✓ Extract files to the proper location
   ✓ Unblock both executables (no manual SmartScreen steps needed)
   ✓ Create desktop and Start Menu shortcuts
   ✓ Optionally enable auto-start with Windows

5. After installation is complete, you can safely delete:
   - The ZIP file (DeviceBatteryTray-vX.Y.Z-win-x64.zip)
   - The extracted folder in Downloads

6. The uninstaller (uninstall.ps1) is copied to your installation directory
   for easy removal later.


Option 2: Manual Installation
───────────────────────────────────────────────────────────────────────────────

1. Extract this ZIP to a permanent location:
   - %LOCALAPPDATA%\DeviceBatteryTray (recommended)
   - C:\Program Files\DeviceBatteryTray\ (requires admin)

2. Right-click DeviceBatteryTray.exe → Properties → Unblock (if shown)
   Right-click LGSTrayHID.exe → Properties → Unblock (if shown)

3. Run DeviceBatteryTray.exe


UNINSTALLATION
───────────────────────────────────────────────────────────────────────────────

Option 1: Using the Uninstaller
───────────────────────────────────────────────────────────────────────────────

1. Navigate to your installation directory:
   - %LOCALAPPDATA%\DeviceBatteryTray, or
   - C:\Program Files\DeviceBatteryTray\

2. Double-click "uninstall.bat" (or right-click → Run as Administrator if needed)

3. The uninstaller will:
   ✓ Stop running processes
   ✓ Remove auto-start registry entry
   ✓ Remove desktop shortcut
   ✓ Remove Start Menu shortcut
   ✓ Delete all installation files


Option 2: Manual Uninstallation
───────────────────────────────────────────────────────────────────────────────

1. Exit DeviceBatteryTray (right-click tray icon → Exit)

2. Remove auto-start (if enabled):
   - Press Win+R, type "regedit", press Enter
   - Navigate to: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
   - Delete "DeviceBatteryTray" entry

3. Delete shortcuts:
   - Desktop: Delete "DeviceBatteryTray.lnk"
   - Start Menu: Delete folder "DeviceBatteryTray" in Programs

4. Delete installation folder:
   - %LOCALAPPDATA%\DeviceBatteryTray, or
   - C:\Program Files\DeviceBatteryTray\


CONFIGURATION
───────────────────────────────────────────────────────────────────────────────

Application Settings (appsettings.toml)
───────────────────────────────────────────────────────────────────────────────

Edit appsettings.toml in the installation directory to configure:

UI Settings:
  • enableRichToolTips: Show detailed tooltips with battery percentage and
    last update time (true/false)

Native Device Manager:
  • enabled: Enable/disable device detection and battery monitoring (true/false)
  • retryTime: Seconds between poll attempts when device is off/unavailable
    (default: 5 seconds)
  • pollPeriod: Seconds between battery polls when device is connected
    (default: 120 seconds / 2 minutes)
  • disabledDevices: List of device name substrings to ignore (case-insensitive)

See appsettings.toml for detailed comments and examples.


Tray Menu Settings
───────────────────────────────────────────────────────────────────────────────

Right-click the tray icon to access:

  • Devices: Select which devices to monitor
  • Rediscover Devices: Manually refresh device detection
  • Display Numeric Icon: Show battery percentage as number instead of icon
  • Autostart with Windows: Enable/disable auto-start on login
  • Check for Updates: Manually check for new versions
  • Download Update: Download and install available update
  • Auto-check for Updates: Enable/disable automatic daily update checks

TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

Problem: Windows SmartScreen blocks the app
Solution: The installer automatically unblocks files. If running manually,
          right-click each .exe → Properties → Unblock

Problem: Device not detected
Solution: 
  1. Try "Rediscover Devices" from the tray menu
  2. Ensure device is powered on and connected
  3. Check if device is in disabledDevices list in appsettings.toml

Problem: App won't start after update
Solution: Ensure both DeviceBatteryTray.exe and LGSTrayHID.exe are unblocked

Problem: Auto-start not working
Solution: 
  1. Check "Autostart with Windows" is enabled in tray menu
  2. Verify registry entry exists:
     HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
     Should contain "DeviceBatteryTray" entry
